9/19: Announcing Cadence/ARIA

The Asynchronous Request Intermediary Application, or ARIA, is the project name of the next major step in Cadence's development. Though the majority of changes up to now have been front-end design, Aria is a completely new back-end addition.

This is all new territory for me, so while development will certainly be slow, it's my way of finally getting around to learning the technologies I've been wanting to. Aria is "Cadence 2.0".

Aria's planned changes include a complete server OS and webserver change, as well as an upgrade to stream source software. A song database, one of the original features of Cadence (though never fully implemented), will be making a return as well. If things seem to work up to this point, a song request application will tie everything together.

View the Aria Branch on GitHub.

About Cadence:

Cadence is an online radio heavily inspired by R/a/dio. Originally started in February 2017, the project is my first endeavour to practice a full range of IT-skills. Development continues today with a focus on back-end technology.

The server typically plays all the genres of music I like, a mix of classic rock, synthpop, and metal. I occasionally override the server and play a certain genre, artist, or the same song on an endless loop.

If you're interested in seeing this project's development, nearly everything is available on GitHub: source/issues.

Cadence Radio is a DMCA compliant, non-commercial webcast made for educational purposes.
